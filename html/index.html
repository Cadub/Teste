Sobre o Findly

Criado por um grupo de estudantes do Instituo Mauá de Tecnologia o Findly se apresenta como uma opção para os estudantes, cuja faculdade é Mauá, de poder realizar perguntas referentes a diferentes matérias de cursos específico. A partir disso, ao lançar a pergunta outros estudantes podem responde-las, de modo a ajudar no melhor entendimento de um assunto

Nossa missão é: "Proporcionar uma melhor interação entre os estudantes e entendimento das matérias".

Oferecemos uma sistema com uma interface simplificada, para que o usuário sinta-se confortável ao utilizar a aplicação Findly, além de apresentar uma mecanica simples para realizar perguntas e responde-las
